# chatgame: Language Models with Game-Theoretic Solvers

## Work in Progress

A lightweight, extensible framework for [steering language models using game-theoretic solvers](https://arxiv.org/abs/2402.01704).

## Installation

   ```bash
   git clone https://github.com/yourusername/chatgame.git
   cd chatgame
   ```

## Usage

  ```bash
   python main.py
   ```
